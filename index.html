Bluetooth Trilateration Prototype

Jan 2015 - Apr 2015

The goal of this project was to explore whether it is possible to trilaterate the location of a mobile device utilizing Bluetooth beacons. By trilaterating a mobile device, we are able to use the coordinate data gathered from the device to to build a heat map to analyze the movement of the users. This project involved working in a team environment to build two main components, an Android app and a node.js web application. The Android app will be used to gather and store coordinate data onto the cloud, and the web application will be used for analyzing the data retrieved from the cloud.
